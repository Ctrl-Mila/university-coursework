# Software Engineering I – Selected Academic Materials

This folder contains selected academic materials from the course **Software Engineering I**
(Analista Programador Universitario – UNLP).

---

## Included materials

### 1. State Transition Diagrams (DTE)

- **Theory**  
  Explanation of the formal model, concepts and notation used to represent system behavior.

- **Practice**  
  Practical exercises focused on modeling system states and transitions based on given requirements.

### 2. Use Cases

- **Practice**  
  Complete set of use case exercises developed during the course, including:
  - applied theoretical overview
  - fully worked example
  - multiple practice exercises with structured tables
