# Parking Management System

Object-Oriented Programming exercise.

This project models a parking lot system that manages cars organized by floors and parking spots.

## Features
- Register a car in a specific floor and parking spot
- Search for a car by license plate 
- Count how many cars are parked in a given spot across all floors 
- Generate a complete textual representation of the parking lot state

## Classes
- `Auto`: represents a car with the owner's name and license plate
- `Estacionamiento`: manages the parking lot structure and operations 
- `Ejercicio2`: main test program that exercises the system
