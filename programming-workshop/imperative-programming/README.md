# Imperative Programming – Programming Workshop

This section contains selected exercises developed during the course
**Programming Workshop** (Analista Programador Universitario – UNLP).

The focus is on fundamental imperative programming concepts implemented in **Pascal**.

## Topics covered

- Sorting algorithms (selection and insertion)
- Recursion
- Binary search trees
- Tree traversals and operations

## Skills

- Algorithmic thinking
- Problem decomposition
- Recursive problem solving
- Data structure manipulation
